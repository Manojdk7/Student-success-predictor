# Student-success-predictor

📖 Overview

This project is a Machine Learning-based Student Success Predictor that determines whether a student is likely to Pass or Fail based on key performance indicators.

🎯 Objective

To analyze student data and build a predictive model that helps identify students at risk, enabling early intervention.

📊 Features Used
Study Hours
Attendance
Past Scores
Sleep Hours
Internet Access (encoded)
⚙️ Technologies Used
Python
Pandas
NumPy
Scikit-learn
Matplotlib
Seaborn
🧠 Machine Learning Approach

Although initially planned as unsupervised learning, the project uses:

Logistic Regression (Supervised Learning)
Data Scaling using StandardScaler
Label Encoding for categorical variables
📈 Model Evaluation
Classification Report (Precision, Recall, F1-score)
Confusion Matrix Visualization
📷 Output Visualization
Heatmap of confusion matrix using Seaborn
💻 How to Run
# Clone the repository
git clone https://github.com/your-username/student-success-predictor.git

# Navigate to project folder
cd student-success-predictor

# Install dependencies
pip install -r requirements.txt

# Run the project
python studentperformance.py
🧪 Example Input
Enter study hours: 6
Enter attendance: 80
Enter past score: 70
Enter sleep hours: 7
✅ Output
Prediction based on input: Pass
